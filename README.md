# 坦克大战

一个纯 HTML + Canvas 编写的经典坦克大战游戏，无需任何依赖，浏览器打开即可游玩。

## 在线游玩

[点击这里在线试玩](https://christyQQ-line.github.io/tank-battle/)

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` / `W` | 向上移动 |
| `↓` / `S` | 向下移动 |
| `←` / `A` | 向左移动 |
| `→` / `D` | 向右移动 |
| `空格` | 射击 |

## 游戏玩法

- 控制绿色坦克消灭所有红色敌方坦克
- 保护底部的金色老鹰基地不被摧毁
- 每关有固定数量的敌人，消灭全部即可进入下一关
- 玩家初始 3 条生命，生命耗尽则游戏结束

## 地形说明

- **棕色砖墙**：可被子弹摧毁
- **灰色钢墙**：不可摧毁
- **蓝色水域**：坦克不可通行
- **绿色草地**：坦克和子弹均可通行，提供隐蔽效果
- **金色老鹰**：基地，被摧毁则游戏结束

## 本地运行

直接双击 `index.html` 在浏览器中打开即可。

```bash
git clone https://github.com/christyQQ-line/tank-battle.git
cd tank-battle
# 用浏览器打开 index.html
```

## 技术栈

- 原生 JavaScript（ES6）
